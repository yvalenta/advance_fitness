Repositorio oficial de la landing page para **Advance Fitness**, un ecosistema de coaching digital y presencial enfocado en hipertrofia, disciplina y resultados reales. Esta página está diseñada para alta conversión, velocidad de carga y posicionamiento SEO local (Medellín/Colombia) e internacional.

---

## 🚀 Tecnologías Utilizadas

Este proyecto fue construido priorizando la velocidad y la simplicidad de mantenimiento, sin necesidad de procesos de compilación complejos (Build steps):

* **HTML5 Semántico:** Estructura accesible y optimizada para SEO.
* **Tailwind CSS (v3 vía CDN):** Framework de utilidades para un diseño a medida y responsivo.
* **Alpine.js (v3 vía CDN):** Framework reactivo ligero para el manejo del estado (selector de planes y mensajes dinámicos de WhatsApp) sin la carga de React o Vue.
* **Lucide Icons:** Iconografía vectorial en formato SVG, ligera y minimalista.
* **Google Fonts:** Bebas Neue (Títulos) y DM Sans (Cuerpo de texto).
